# 🎮 Simon Game

## 📌 About
Simon Game built with HTML, CSS, and JavaScript. A memory-based game where players repeat an increasingly long sequence of colors generated by the system. Each level adds a new step, making the game more challenging.

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 🎯 Features
- Color sequence memory game
- Increasing difficulty levels
- Interactive UI design
- DOM manipulation

## 🚀 How to Play
1. Press Start to begin the game 
2. Watch the sequence of colors  
3. Repeat the sequence in the correct order  
4. Each level adds a new color  
5. If you make a mistake, the game restarts

## 👩‍💻 Project Purpose
This project was built to practice JavaScript logic, DOM manipulation, and interactive user interface design.
